#Password Strength Analyzer Web App
## Overview
This is a Flask-based web application that analyzes password strength and entropy in real time. It provides users with instant feedback on password security.

## Features
Web interface for password input
Real time strength analysis
Entropy calculation
User-friendly UI

## Security Concepts
Password entropy
Authentication security
Brute-force resistance analysis

## Technologies Used
Python
Flask
HTML/CSS

## Security Analysis
This tool demonstrates how weak passwords can be vulnerable to brute force attacks.
Passwords with low entropy can be cracked quickly using modern computing power, while stronger passwords significantly increase attack resistance.
